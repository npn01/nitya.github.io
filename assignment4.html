Assignment 4: Piston Unit

For this assignment, I created a simple piston with 3 parts & 2 fastners.


Source Files

AI working file


Inspired by

BirdBrain Technologies - Piston Basic Mechanism

Printer settings

Laser Printing settings: 100% power, 20% speed, 50% freq.

I browsed many CAM designs, & basic mechanisms. In the end, I decided to go with a simple piston design.

Prototype

I first created created a cardboard prototype by cutting out the 3 rectangles and added holes for the fastner.

Laser printing

I used a wood panel with 2.5mm thickness for the laser printing. I created interwoven patterns so that the 4 sides would nest within each other.

I created 4 rectangular frames with different patterns.

Final Output

Image of the final lamp design.

Issues encountered & lessons learnt

I created 4 slots in the base shape to insert the 4 wooden panels. I printed the 3D base first, however, the wooden panel measurements didn't line up. As a result I couldn't put the panels inside the slot. If I was redoing this I would be sure to get exact measurements before cutting the panel.
